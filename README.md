## Rock Paper Scissors Game
### 项目要求
剪刀石头布网格对战游戏，后端使用Node.js，前端使用Electron封装。

### ✅项目运行
1. 下载rep到本地。
2. 打开终端，cd到该项目。
3. 输入'npm start'。
